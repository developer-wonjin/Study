# 큐



## 1. 성질

| --                                                           | 시간복잡도 | --   |
| ------------------------------------------------------------ | ---------- | ---- |
| 원소의 추가                                                  | O(1)       |      |
| 원소의 제거                                                  | O(1)       |      |
| 제일 앞(front) / 뒤(rear) 원소확인                           | O(1)       |      |
| 제일 앞/뒤가 아닌 나머지 원소들의 확인/변경이 원칙적으로 불가능 |            |      |

- push / pop  뒤쪽으로 추가/제거 실행
- front/ back 앞/뒤 조회

const int MX = 1000005;
