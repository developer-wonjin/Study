## 2.2 안전하지 않은 문자

### 안전한 전송이란

- 정보를 유실하지 않는 것

### URL이 허용하는 문자

- ASCII 코드에 속하는 알파벳문자
- 이진데이터나 ASCII코드 바깥영역은 인코딩하여 %와 알파벳으로 변형하여 안전한 문자로 변경하여 전송
